Page Layout Techniques

using semantic tags for layout
Semantic tags are those tags clearly defines its meaning to both the browser & the developer
Eg: < header>, < main> and < footer>
While non-semantic tags are tags that tell nothing about its content.
Eg: < div> and < span>
